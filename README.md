# 🚗 WashPro – Smart Car Wash Management System

A full-stack Car Wash Management System built using Spring Boot to streamline operations for Admin, Employees, and Customers. The system enables efficient vehicle management, service tracking, and automated billing.

---

## ✨ Features

- 👤 Role-based system (Admin, Employee, Customer)
- 🚗 Vehicle registration & management
- 🛠 Service selection (Daily, Weekly, Alternate plans)
- 📍 Location-based insights
- 🔄 RESTful APIs for all operations

---

## 🛠 Tech Stack

- **Backend:** Java, Spring Boot, Spring Core
- **Database:** MySQL, MongoDB
- **ORM:** JPA, Hibernate
- **Architecture:** REST APIs, MVC Design Pattern
